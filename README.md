**CHATTRIX**

A Simple & Efficient Real-Time Chat Application in Java

CHATTRIX is a lightweight real-time chat application built using Java Sockets and Multithreading.
The goal of this project is to understand how real-world messaging systems work at a core networking level, before scaling it to web and mobile platforms.

This project is designed to grow step-by-step â€” from a basic socket chat to a WhatsApp-like system.

**Features (Current Stage)**
* Real-time messaging using Java Sockets
* Multi-user chat support
* Multithreaded server (handles multiple clients simultaneously)
* Simple Swing-based GUI
* Clean and minimal codebase

**Tech Stack**

* Language: Java
* Networking: TCP Sockets
* Concurrency: Multithreading
* GUI: Java Swing
* IDE: IntelliJ IDEA
